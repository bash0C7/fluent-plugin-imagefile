# fluent-plugin-imagefile

## Output

Output plugin to save image file from massages attribute value

### Configure Example

````
    type iamgefile
    save_dir SAVE_DIR
    filename_key filename_field
    extension jpg
    image_key image_field
    base64encoded false
````
